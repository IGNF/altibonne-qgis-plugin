# Altibonne

Altibonne is a Python plugin that allows visualizing and correcting elevation profiles on linear features (max 2 contiguous) or surface features (max 1)

## Pré-requis

Mandatory : The "plugin\_maitre" must be installed.
Link to the plugin maître : [maitre-qgis-plugin sur GitHub](https://github.com/IGNF/maitre-qgis-plugin)

## Fonctionnalités

French : 
* Visualisation d'un profil
* Modification des tous les z d'un tronçon et/ou des sommets sélectionnés
* Modifier ponctuellement un Z d'un point
* détecter des pentes suivant un seuil

English :
* Viewing a profile
* Modifying all the zs of a segment and/or the selected vertices
* Modifying a single Z of a point
* Detect slopes according to a threshold
